##### # "구적도","건물배치도" "공사계획 평면도" 
    - 리스트의 허가용도, 허가면적 등... 값 변경후 다시 에디팅시 데이터 유지 안됨
    - 클릭시 에디팅 모드로 동작하게.
        - 에티팅후 다른 에디팅 영역 클릭시 에디팅 모드로 들어가지 않고, 기존 영역이 에디팅에서 해제되는 상황인데
        - 클릭시마다 에디팅이 활성화되게.
    - 제외지 등  NaN이 오고 있는지? 고칠 수 있다면 NaN가 표시 안되게.
